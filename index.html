Watched Videos On YouTube Remover

YouTube only allows you to remove all watched videos from the Watch Later playlist, not taking progress into account. Videos that have only been partially viewed will also be deleted from the playlist.

★ By means of this extension, the fully viewed videos can be deleted automatically from the playlist.

★ The partially watched videos remain in the playlist and can be watched at a later time.

The extension extends the action menu by the entry "Remove completely watched videos".

"The 'Watch Later' playlist on YouTube employs a lazy loading technique, meaning that it loads content as needed rather than all at once. Therefore, this extension operates based on the entries that have already been loaded. It doesn't automatically load all entries in the 'Watch Later' playlist. For the extension to consider more videos, you would need to manually scroll down through your playlist to load more entries."

onvista Json Stock Url Finder for Portfolio Performance

Mit dieser Erweiterung können Sie die notwendigen Informationen zur Darstellung der historischen Daten (JSON) in Portfolio Performance direkt auf der OnVista-Website anzeigen lassen.

Hauptmerkmale

★ Aktivierbarer Schalter: Die Erweiterung verfügt über einen Schalter, mit dem Sie die Funktionalität je nach Bedarf aktivieren oder deaktivieren können.

★ Dialoganzeige: Sobald ein neuer Chart auf onvista angefordert wird und die Funktion aktiviert ist, bspw. durch Klick auf ein neues Zeitintervall, zeigt die Erweiterung einen Dialog mit den zur Darstellung der historischen Daten in Portfolio Performance notwendigen Informationen an.

★ Einfache Integration: Die in dem Dialog angezeigten Informationen können einfach kopiert und in Portfolio Performance eingefügt werden.
